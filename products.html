Home Screen Interface Customization

This idea is centered on having a simple interface which is easy for the user to have access and navigate while giving them freedom of customization. Assuming that the user has more than one sections of their hydroponic system or even others at various locations, the user can categorize their system using the customization feature so that they can easily navigate and find out which sector or system they want to know about. User can change the name and add icons as well to have better representation for each part their system or another one at a different location.

Set Hydroponics Schedule Automation

The scheduling system is a feature where the user will be able to design an automated system for their hydroponic design. Users will be able to select a time of day to turn their system on or off, for example; hydroponic system will be active from 6a.m and will become inactive at 6p.m.

Check Temperature information Data Collecton

With the use of sensors the temperature reading is taken and sent to the rasberry and then to the smart phone. This gauge will display on the app and it will range from low to high along with an added feature where if the user wants to see additional information that is more precise, they can choose to do so.
